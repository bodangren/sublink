# Track: Scaffold SubLink PWA

## Overview
Initialize the SubLink PWA project using Vite, React 19, and TypeScript. Set up the core shell and offline-first PWA configuration.

## Requirements
- Vite-based React project (TypeScript).
- PWA manifest and service worker configuration.
- Basic routing (Home, Compliance, Tasking).
- Initial "Rugged" CSS theme.

## Architecture
- **Frontend:** React with Vite.
- **Offline:** 'vite-plugin-pwa'.
- **Theme:** High-contrast CSS custom properties.
