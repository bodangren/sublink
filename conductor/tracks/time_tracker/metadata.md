# Track: Time Tracker

## Metadata
- **Track ID:** time_tracker
- **Created:** 2026-03-08
- **Status:** In Progress
- **Priority:** High
- **Model:** glm-5

## Summary
Add time tracking functionality to allow subcontractors to track billable hours on projects and tasks. Features include a live timer, manual entry, and time summaries.

## Scope
- IndexedDB schema for time entries
- Timer component with start/stop
- Time entry form (manual entry)
- Time entry list view
- Dashboard widget showing today's hours
- Integration with existing projects

## Out of Scope
- Cloud sync (future track)
- Invoice generation from time entries
- Export to external systems
